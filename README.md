# Enlaces Publicitarios - Panel Administrativo

Panel administrativo en Angular con login JWT básico, layout con header/footer y sidebar colapsable, usando Bootstrap.

## Requisitos

- Node.js 20+
- npm

## Desarrollo

- `npm install`
- `npm run start`

Abre http://localhost:4200/

## Build

- `npm run build`

## Estructura principal

- `src/app/layouts`: layouts por rol (admin/cliente)
- `src/app/auth`: login, guard y servicio de autenticación
- `src/app/pages`: vistas del panel (dashboard)
